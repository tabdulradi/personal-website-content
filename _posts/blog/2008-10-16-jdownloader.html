JDownloader … One of the best Download Manager I’ve Ever used

It makes rapidshare much easier ( and megaupload too )

It doesn’t take the links in a single line inputbox, but in a BIG multiline textarea

each link in a line … OK ? it matches the links and get the common part , this will the Folder’s name ;)

and add the files to it’s downloading queue , and begin downloading .. SMARTLY

after the first file downloads .. it wait the 15 minutes between each file , then start the next file

.. and you can give the program your router’s username and password , so when you reach rapidshare’s limit it automatically reset your connection , to renew your IP , then continue downloading !!!!

In case you are downloading RARed Parts with password , you can type the password, so the program will automatically extract the files after it finishes the download !

Begad … WOW
